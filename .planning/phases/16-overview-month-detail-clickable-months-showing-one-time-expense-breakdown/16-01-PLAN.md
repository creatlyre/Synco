   Add a chevron span (▸) in the month name cell. On click, toggle a detail row below.

   If clicking different month, collapse previous and expand new.

2. **Detail row**: After each month `<tr>`, render a hidden `<tr class="detail-row">` with
   `colspan="7"`. This row contains:

     delete button (✕)

     `PUT /api/budget/expenses/{id}` with ExpenseUpdate, then `reload()`
